Frequently Asked Questions

Q: How much does Azure SmartCost cost?

A: Free to deploy. You only pay for Azure infrastructure costs (estimated $20-50/month).

Q: What data does SmartCost access?

A: Only cost data from Azure Cost Management API (read-only). No resource data, VM contents, or sensitive information.

Q: Why is the dashboard not showing costs?

A: Ensure "Cost Management Reader" role is assigned. Wait 5-10 minutes after deployment for initial sync.
